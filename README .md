# 🍎 Apple Leaf Disease Detector

An AI-powered web application that detects apple leaf diseases using Deep Learning and provides intelligent care recommendations.

---

## 🚀 Features

* 📷 Upload an image of an apple leaf
* 🧠 Predict disease using a trained CNN model
* 📊 Display confidence score
* 🤖 Get AI-generated treatment advice

---

## 🧠 Tech Stack

* **Frontend:** Streamlit
* **Model:** TensorFlow / Keras (CNN)
* **Libraries:** NumPy, Pillow
* **LLM Integration:** LangChain + OpenRouter

---

## ▶️ Run Locally

```bash
git clone https://github.com/your-username/apple-leaf-disease-detector.git
cd apple-leaf-disease-detector
pip install -r requirements.txt
streamlit run app.py
```
